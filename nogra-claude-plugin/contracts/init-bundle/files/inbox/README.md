# inbox — the shared desk

Two-way. The one place you and Nogra hand things back and forth.

```
inbox/
├─ screenshots/   you → Nogra   the newest image is what "tjek" reads
├─ drops/         you → Nogra   files, links, raw material to pick up
└─ out/           Nogra → you   receipts, drafts, "ready for GO"
```

**"tjek inbox" works both ways.** You say it → Nogra reads the newest thing you
dropped. Nogra says it → look in `out/` for what's ready for your eyes.

`out/` is a review tray, not a done tray — anything there is waiting for you to
look, decide, or give GO. Nothing acts on its own.
